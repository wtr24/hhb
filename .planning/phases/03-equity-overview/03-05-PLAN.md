     - `sigma = row["impliedVolatility"]`
     - `greeks = bs_greeks(S, row["strike"], T, r, sigma, "call")`
     - Build dict: `{"strike", "bid", "ask", "lastPrice", "volume", "openInterest", "iv": sigma, "delta", "gamma", "vega", "theta"}`
   - Same for `chain.puts` with `option_type="put"`
   - Compute IV surface grid: for each of the 5 expiries, fetch chain and extract IV values across strikes. Build matrix: `{"strikes": [...], "expiries": [...], "iv_matrix": [[...], ...]}`
   - Compute IV percentile rank: collect all IV values from the nearest-expiry chain, `rank = iv_percentile_rank(median_iv, all_ivs)` — rough rank across strikes (comment noting 52-week rank is Phase 8)
   - Return:
     ```json
     {

       "expiry": "2026-04-25",
       "expiries": ["2026-04-25", ...],
       "current_price": 189.50,
       "iv_rank": 73.0,
       "calls": [{strike, bid, ask, lastPrice, volume, oi, iv, delta, gamma, vega, theta}, ...],
       "puts": [{...}],
       "iv_surface": {"strikes": [...], "expiries": [...], "iv_matrix": [[...], ...]},
       "source": "yfinance+black_scholes"
     }
     ```
   - Cache with 15m TTL: `cache_set(redis_client, f"options:{ticker}", response, "quote")`
